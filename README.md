# Nuxt Minimal Starter

Look at the [Nuxt documentation](https://nuxt.com/docs/getting-started/introduction) to learn more.

## Nuxt 기본 폴더 구조

```
/assets      # 이미지, 스타일 등 정적 리소스
/components  # Vue 컴포넌트
/layouts     # 레이아웃 컴포넌트
/middleware  # 미들웨어
/pages       # 라우트 페이지
/plugins     # 플러그인
/public      # 정적 파일
/store       # Pinia 등 상태관리(옵션)
```
