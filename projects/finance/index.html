금융권 웹사이트

프로젝트 개요

웹 접근성과 웹 표준을 준수한 금융권 웹사이트 퍼블리싱 프로젝트입니다. 사용자 경험을 최우선으로 고려하여 설계되었으며, 모든 사용자가 쉽게 이용할 수 있도록 접근성을 높였습니다.

주요 기능

  • 웹 접근성 지침 준수 (WCAG 2.1)
  • 반응형 웹 디자인 구현
  • 크로스 브라우징 지원
  • 시맨틱 마크업 적용
  • 성능 최적화

사용 기술

HTML5
CSS3
JavaScript
SCSS

주요 도전 과제

  • 웹 접근성 구현

    스크린 리더 사용자를 위한 ARIA 레이블 적용 및 키보드 네비게이션 구현

  • 성능 최적화

    이미지 최적화 및 코드 스플리팅을 통한 페이지 로딩 속도 개선
